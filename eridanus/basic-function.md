
# 内置功能

该页面是日奈ᴮᴼᵀ的简单功能的集合。

> [!TIP]
> 指令无需@机器人触发，文档写的什么就发送什么来使用功能。


## 天气查询

```yaml
查天气{深圳}          #简单的天气查询
```


## 涩图功能

```yaml
/setu {数量} {tag1} {tag2}      #比如 /setu 3 白丝 萝莉
```
需要授权权限，否则不可使用。


## 角色识别

```yaml
识别 #然后发送图片，一起发也行
```
